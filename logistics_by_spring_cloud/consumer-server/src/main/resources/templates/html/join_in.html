加盟天天

天天快递提供快捷、安全、贴心、可靠的快递服务，是您的优质选择

加盟须知

在认同天天快递品牌及经营管理模式前提下，符合加盟资质的网点经总部同意，独立与天天快递有限公司签订《特许经营加盟合同》，专一经营天天快递品牌，独立投资，自负盈亏，独立对外承担责任；接受总部统一管理，共同致力于提升客户满意度，提高服务品质，创造和谐可持续发展的网络环境。
遵守法律法规，依法经营。
企业法定代表人（职业经理人）具有高中以上学历，有一定的经济实力和经营管理能力。
经营资质健全，依法注册有限责任公司，具备快递、寄递、或速递服务项目的资质，符合邮政行业标准要求，且取得快递业务经营许可证书。
具有固定的经营场所，包括办公场所、快件存放场所、中转仓库等，员工生活区与工作场地必须分开。
有适合业务公司发展需要的从业人员，包括但不限于管理人员、业务员、客服及仓库管理人员。
有必要的办公设备和硬件设备，包括电脑、传真、电话、复印机、巴枪、监控、消防设施等，保障公司日常工作开展。
有足够的车辆承担快件的收取、派送和中转运能。
加盟网点按照省级城市、地级城市和县区级城市的等级进行分类，加盟费用则按照等级分类标准收取，加盟网点应缴纳押金、网络建设费、商标使用费。
能够妥善协调国家相关职能部门的关系，保证快件在所加盟区域内安全中转与送达。

加盟流程

线下提交申请
凡符合加盟条件的快递公司、企业或者个人，有资金投入且具有快递经营资格，均可向总部提出申请加盟。
审核
总部会派人实地考察申请人在该经营地区的资质与条件，如符合条件，由所在地区上属管理片区的经理/专员初步审核经营方案细则和派送区域，达成共识后，上报总部进行报批。
签约
流程报批完成，天天快递有限公司同加盟网点签订《天天快递网络特许经营加盟合同书》（下面简称“合同”），合同一经签订，双方享受合同条款的权力与义务，共同致力于健康的网络建设。
